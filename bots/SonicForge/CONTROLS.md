# 🎵 SonicForge: The Music Bot 🎵

**Role**: High-Fidelity Audio Streaming & Forge Ambience.

## 🎸 Features
- **Hi-Fi Stream**: 128kbps+ audio quality.
- **Multi-Source**: Integration with Spotify, SoundCloud, and local Forge files.
- **Ambience Mode**: Lo-fi coding beats (24/7 "The Forge" radio).

## 🎮 Commands & Controls
| Command | Restricted | Description |
| :--- | :--- | :--- |
| `!play <url/search>` | No | Add a track to the current queue. |
| `!forge` | No | Switch to 24/7 Lo-fi Coding Beats mode. |
| `!lyrics` | No | Extract and display AI-synchronized lyrics. |
