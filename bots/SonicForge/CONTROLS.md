# 🎵 SonicForge: High-Fidelity Music Bot
=======================================
SonicForge handles the Academy's audio environment.

### 🎮 Features
- **HQ Audio**: Low-latency 24/7 streams.
- **Playlists**: Collaborative playlist management.
- **AI Rhythm**: Smart track selection based on community mood.

### ⌨️ Commands
- `!forge <search>`: Search and play a track in the forge.
- `!playlist`: View the current Academy queue.
