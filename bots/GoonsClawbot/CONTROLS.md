# ⚔️ GoonsClawbot: Operations & Controls ⚔️

**Role**: Global Server Supervisor & Onboarding Mentor.

## 🛠️ Performance & Specs
- **Engine**: Python 3.12 (Native)
- **AI Core**: Google Gemini 1.5 Flash (Distilled)
- **Context Window**: 1M+ Tokens for Deep Project Tracking.
- **Onboarding Logic**: Multi-choice Questionnaire + "I AGREE" Modal Gate.

## 🎮 Commands & Controls
| Command | Restricted | Description |
| :--- | :--- | :--- |
| `!ask <query>` | No | AI Mentorship & Code Review. |
| `!sync` | Admin | Manually sync logs to the Brain. |
| `!status` | No | Display live bot metrics & server health. |
| `!tiers` | No | Show "Noob to God" progression path. |

## ⚙️ Maintenance
- **Logs**: Located in `~/.openclaw/logs/goonsclawbot`
- **Session Data**: Encrypted in sqlite3 database `goons_brain.db`.
