# Photopea Design System for Apptivators Academy

Inspired by the sleek, utility-focused, and high-performance design of [Photopea](https://www.photopea.com/).

## Visual Tokens

### Color Palette
- **Forge Charcoal**: `#1e1e1e` (Primary backgrounds)
- **Academy Gold**: `#ffd700` (Tier 5 / God highlights)
- **Sentinel Red**: `#e74c3c` (Security alerts / SAMPI.RT)
- **Matrix Green**: `#2ecc71` (Verified success)

### UI Components
- **The Brain Expansion**: Use `Brain1.png` through `Brain1-5.png` as sequence headers in `#roles`.
- **White Hat Banners**: Custom banners generated in Photopea with the "Forge" theme for channel headers.
- **Micro-interactions**: Use custom emojis from the `Emoji's/` directory for button labels.

## Onboarding Flow (The User Journey)

1. **Gate 0: #welcome**
   - Header: `Apptivators Academy.png`
   - Interaction: Click ⚔️ to enter.

2. **Gate 1: #rules**
   - Interaction: Read and Interacted with "I AGREE" box.

3. **Gate 2: #roles**
   - Header: `Brain1-5.png`
   - Interaction: Questionnaire bot prompts for experience level.

4. **Final: The Black Box**
   - User types "I AGREE" in all caps.
   - SAMPI.RT grants access token.
