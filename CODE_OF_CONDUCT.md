# 📜 Apptivators Academy: Code of Conduct & Rules

## ⚔️ The Founding Principle
**"One App At A Time."** We are a community of builders, protectors, and mentors. Our rules are designed to ensure the forge remains hot and the edge remains sharp.

## 1. Zero Tolerance for Malware
Security is our highest protocol.
- **Malware Snapshot**: Any code flagged as malicious by 3 Admins/Mods triggers an instant snapshot of the user's ID for permanent logging.
- **Immediate Removal**: Flagged code is purged immediately to protect the community.

## 2. Open Source & Educational Integrity
- **Public = Shared**: Any code posted in public channels is considered open-source for community learning.
- **Documentation**: All code shares should include references (GitHub, GitLab) or clear claims of authorship.

## 3. Professionalism & Multiculturalism
- **No Harassment**: Zero tolerance for bullying, hate speech, or discrimination.
- **Multicultural Space**: Personal worldviews (politics, religion) should be kept private to maintain a creative environment focused on code.

## 4. Fair Review & Ban Protocols
- **Case Files**: All bans are recorded under a unique Ban File Number.
- **Right to Appeal**: Banned members have one chance to explain their actions before a final decision.
- **Full Disclosure**: Final ban reports are shared with the user via a valid contact method.

## 5. The "Blackout Rule" (🔒 Private Channels)
- **Total Purge**: Code in private review chambers is sensitive. When all members exit a private voice/review channel, it is automatically purged of all data.

💯 *By participating in this server, you agree to uphold these legendary standards.*
